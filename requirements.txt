# ========================
# Installation Notes
# ========================
# 1. Create virtual environment:
#    python -m venv venv
#
# 2. Activate virtual environment:
#    Windows: venv\Scripts\activate
#    Linux/Mac: source venv/bin/activate
#
# 3. Install dependencies:
#    pip install -r requirements.txt
#
# 4. First run will download ~2GB of models:
#    - SentenceTransformer: all-MiniLM-L6-v2 (~80MB)
#    - Google Gemma 2B-Instruct (~4.5GB)
#
# 5. Verify installation:
#    python -c "import torch; import transformers; import qdrant_client; print('OK')"

# ========================
# Known Issues
# ========================
# Issue: torch installation fails on Windows
# Solution: Install Visual C++ Build Tools
# https://visualstudio.microsoft.com/visual-cpp-build-tools/
#
# Issue: Out of memory loading Gemma 2B
# Solution: Ensure 8GB+ RAM available or use smaller model
#
# Issue: qdrant-client connection timeout
# Solution: Check firewall settings and QDRANT_URL in .env
